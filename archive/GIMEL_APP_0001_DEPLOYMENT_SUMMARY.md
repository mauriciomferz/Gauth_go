### **Quick Start**

   ```bash
   git clone https://github.com/Gimel-Foundation/Gimel-App-0001.git
   cd Gimel-App-0001
   git checkout gimel-app-0001-deployment
   ```

2. **Start Backend Server**:

   ```bash
   cd gauth-demo-app/web
   python3 -m http.server 3000
   ```

4. **Access Application**:
